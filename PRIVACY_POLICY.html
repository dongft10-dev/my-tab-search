Privacy Policy - MyTabSearch

MyTabSearch 扩展不会收集、存储或传输任何个人数据到外部服务器。

本扩展仅在用户本地浏览器中运行，所有数据处理均在用户设备上完成。

1. Data Collection

MyTabSearch extension does not collect, store, or transmit any personal data to external servers.

This extension runs entirely in user's local browser, and all data processing is completed on user's device. We do not collect:

  • Browsing history
  • Search query content
  • User identity information
  • Device information
  • Location information
  • Any other personal data

2. Data Usage

This extension uses the following permissions only to implement core functionality:

2.1 Tabs Access (tabs)

  • Purpose: Read and manage browser tab information
  • Data Types: Tab title, URL, tab ID
  • Use Cases:
    • Display all open tabs in the search box
    • Switch to the user-selected tab
    • Close the user-specified tab
    • Track tab activation and closure events to maintain history

2.2 Website Icons (favicon)

  • Purpose: Display website icons
  • Data Types: Website icons
  • Use Cases: Display website icons for each tab in search results to help users quickly identify them

2.3 Local Storage (storage)

  • Purpose: Save user settings and state
  • Data Types: Language preference, previous active tab ID, tab history
  • Use Cases:
    • Save user's language preference settings
    • Save the ID of the previously active tab for quick switching functionality
    • Save tab history to support "switch to previous tab" functionality

2.4 Active Tab (activeTab)

  • Purpose: Access the currently active tab
  • Data Types: Current tab information
  • Use Cases: Get current active tab information to work with other permissions for tab switching functionality

3. Data Storage

All data is stored only in the user's local browser's chrome.storage:

  • Data is stored on the user's device
  • Not uploaded to any server
  • Users can clear this data at any time
  • Data is not synced to other devices (unless the user has enabled browser sync functionality)

4. Data Sharing

MyTabSearch extension does not share user data with any third parties.

We do not:

  • Sell user data
  • Provide data access to third parties
  • Use data for ad targeting
  • Use data for any purpose outside of extension functionality

5. Data Security

Although this extension does not collect or transmit data, we still take the following measures to protect user privacy:

  • All data processing is completed locally on the user's device
  • No network connections are established (except for necessary browser connections)
  • No third-party analytics or tracking services are used
  • Code is open source, and users can review all functionality implementations

6. User Rights

Users have the following rights:

6.1 Access Data

Users can view data stored in the extension at any time:

  • Open Chrome browser
  • Visit chrome://extensions/
  • Find MyTabSearch extension
  • Click "Details"
  • View stored data in "Extension options"

6.2 Delete Data

Users can delete all data stored by the extension at any time:

  • Click "Clear storage data" on the chrome://extensions/ page
  • Or uninstall the extension to delete all data

6.3 Uninstall Extension

Users can uninstall the extension at any time:

  • Click "Remove" on the chrome://extensions/ page
  • After uninstalling, all stored data will be deleted

7. Children's Privacy

This extension is not designed for children under 13 and does not intentionally collect personal information from children. If we discover that we have inadvertently collected information from children, we will delete it immediately.

8. International Users

This extension supports multiple languages (English, Chinese) and respects privacy regulations in different regions:

  • EU Users: Complies with GDPR (General Data Protection Regulation)
  • California Users: Complies with CCPA (California Consumer Privacy Act)

Since this extension does not collect user data, there is no need to provide data export or deletion request processing processes.

9. Policy Updates

We may update this privacy policy from time to time. For significant changes, we will notify users in the extension update notes. We recommend that users review this policy regularly.

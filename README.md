# my-tab-search


![输入图片说明](images/image.png)
## 描述

Chrome浏览器标签管理扩展工具，支持快速搜索关键字并切换到相应的标签。支持使用上下方向键切换所选的选项卡条目，并通过按enter键确认跳转到目标页面。


## 特性

- 根据标题搜索和过滤选项卡。
- 快速访问相关选项卡。
- 键盘快捷键支持，方便切换。
- 支持使用上下方向键切换所选的选项卡项。
- 支持键盘按Delete键，移除已选中的tab条目。

## 安装

1. 发行版列表中下载crx文件；
2. 到浏览器的扩展管理页面“chrome://extensions/”，启用右上角的“开发人员模式”；
3. 将crx拖拽到浏览器中，完成安装。

## 使用

- 通过单击Chrome工具栏中的扩展图标打开扩展弹出窗口，或使用默认的键盘快捷键(ctrl + Shift + A，首选!)
- 在输入框中输入搜索查询，过滤选项卡。
- 单击列表中的选项卡或按“向上”或“向下”方向键选择目标项目，按“Enter”键切换到目标项目。


## 快捷键

1. 呼出tab列表快捷键是Ctrl+Shift+A
2. 快速切换到前一个打开的tab快捷键是 Ctrl+Shift+S （since 1.6.0)
3. 若发现默认快捷键功能无效，可能是按键冲突了，可以到【chrome://extensions/shortcuts】中自己调整一下即可。
4. 为了更流畅使用，建议安装好后，直接固定到浏览器面板上去。

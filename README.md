# my-tab-search


![输入图片说明](images/image.png)
## 描述

Chrome浏览器标签管理扩展工具，支持快速搜索关键字并切换到相应的标签。支持使用上下方向键切换所选的选项卡条目，并通过按enter键确认跳转到目标页面。


## 特性

- 根据标题搜索和过滤选项卡。
- 快速访问相关选项卡。
- 键盘快捷键支持，方便切换。
- 支持使用上下方向键切换所选的选项卡项。

## 安装

1. 克隆或下载此存储库。
2. 打开Google Chrome浏览器。
3. 导航到“chrome://extensions/”。
4. 启用右上角的“开发人员模式”。
5. 点击“加载解压”，选择扩展目录。

## 使用

- 通过单击Chrome工具栏中的扩展图标打开扩展弹出窗口，或使用默认的键盘快捷键(ctrl + Shift + A，首选!)
- 在输入框中输入搜索查询，过滤选项卡。
- 单击列表中的选项卡或按“向上”或“向下”方向键选择目标项目，按“Enter”键切换到目标项目。
- 支持键盘按Delete键，移除已选中的tab条目。

## 快捷键

切换扩展的默认键盘快捷键是“Ctrl+Shift+A”。

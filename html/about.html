MyTabSearch

Version: 1.8.0

MyTabSearch is a Chrome extension that helps you quickly search and switch between your open tabs. With a simple keyboard shortcut, you can access all your open tabs and find the one you need in seconds.

Features:

  • Quickly search through all open tabs
  • Keyboard navigation support
  • Fuzzy search algorithm for better results
  • One-click tab switching
  • Close tabs directly from the search results
  • Support for switching to the previous tab
  • Remember last active tab state

Shortcuts:

  • Ctrl+Shift+A (Cmd+Shift+A on Mac): Open the tab search popup
  • Ctrl+Shift+S (Cmd+Shift+S on Mac): Switch to the previous tab
  • Arrow Up/Down: Navigate through search results
  • Enter: Switch to the selected tab
  • Delete: Close the selected tab

Usage Tips:

  • The shortcut key to quickly switch to the previously opened tab is Ctrl+Shift+S (since v1.6.0).
  • If the default shortcut keys fail to work, a key conflict may be the cause. You can adjust them manually in chrome://extensions/shortcuts.
  • For a smoother user experience, it is recommended to pin the extension to the browser toolbar directly after installation.
